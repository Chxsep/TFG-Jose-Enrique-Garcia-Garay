# TFG-Jose-Enrique-Garcia-Garay
Trabajo de Fin de Grado en Física centrado en la determinación del parámetro de ecuación de estado de la energía oscura (w) mediante el análisis de supernovas de tipo Ia. Utiliza los catálogos SCP Union2.1 y Pantheon+SH0ES para contrastar modelos cosmológicos y evaluar la consistencia del modelo ΛCDM.
